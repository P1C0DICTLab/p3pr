# p3pr7
轻量级3D点云可视化Python库，内置独立离线HTML渲染器，无需外部服务。

## 安装 / Installation / インストール / Installation
### 中文
```bash
# HTTPS 安装主分支最新版
pip install git+https://github.com/P1C0DICTLab/p3pr.git

# 安装指定版本分支
pip install git+https://github.com/P1C0DICTLab/p3pr.git@v7.0
```
完整API、示例与进阶用法请查看仓库内文档：`docs/HowToUse.md`
